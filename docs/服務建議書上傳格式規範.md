# 服務建議書上傳格式規範（璽墨專案管理系統）

供撰寫服務建議書／計畫書時使用。照此格式完成後，上傳至系統可自動帶入**標案名稱、工作項目、合約金額**。

範本檔案：`templates/服務建議書上傳格式範本.docx`

---

## 一、不可變動（固定格式）

以下欄位名稱、結構與寫法請**務必保留**，否則系統可能無法辨識。

### 1. 封面

| 項目 | 固定寫法 | 範例 |
|------|----------|------|
| 計畫名稱 | 前後必須有 **「」** | 「115年高雄市農村加值與產品優化計畫」 |
| 案件類型 | 須含 **委託專業服務案** 或 **委託勞務案** | 委託專業服務案（SP11505） |

```
【機關名稱】（選填）

「【計畫名稱】」
委託專業服務案（【案號】）
服務建議書
```

### 2. 經費章節標題

正文須有**獨立章節標題**（目錄不算），擇一使用：

- `經費預算配置`（建議）
- `經費預算`
- `詳細價目表`

可寫成：`第陸章  經費預算配置`（章號可變，但標題文字須含上述關鍵字）

### 3. 經費表格（最重要）

| 規則 | 說明 |
|------|------|
| 表格類型 | 必須是 **Word 原生表格**（插入 → 表格） |
| 禁止 | **不可**貼截圖、掃描圖、PDF 轉圖 |
| 表頭欄 A | 須含 **`服務細項`**（或 `服務項目`／`工作項目`／`項目名稱`） |
| 表頭欄 B | 須含 **`複價`**（或 `金額`／`預算`／`小計`） |
| 序號 | 建議保留 `序號` 欄，以 1、2、3… 編號 |
| 總計列 | 最後一列須有 **`總金額`** 及合約總額 |
| 金額格式 | 半形數字＋千分位，例 `150,000`、`5,000,000` |
| 禁止 | 金額勿只寫中文大寫（例：伍佰萬元） |

**標準表頭（建議直接沿用）：**

| 序號 | 服務細項 | 單價 | 數量 | 單位 | 複價 | 說明 |
|------|----------|------|------|------|------|------|

**資料列範例：**

| 序號 | 服務細項 | 單價 | 數量 | 單位 | 複價 | 說明 |
|------|----------|------|------|------|------|------|
| 1 | 成立輔導團隊並訂定輔導機制 | 150,000 | 1 | 式 | 150,000 | |
| 2 | 委託輔導費 | 3,200,000 | 1 | 式 | 3,200,000 | |
| 3 | 移地農村產業培訓交流 | 350,000 | 1 | 式 | 350,000 | |
| | **總金額** | | | | **5,000,000** | 含稅 |

### 4. 系統對應關係

| 文件欄位 | 系統欄位 |
|----------|----------|
| 封面「計畫名稱」＋委託專業服務案 | **標案名稱** |
| 服務細項（每一列） | **工作項目名稱** |
| 複價（每一列） | **工作項目可用金額** |
| 總金額 | **合約金額** |

## 二、可變動（內容依案件調整）

| 項目 | 說明 |
|------|------|
| 機關名稱 | 封面可寫可不寫 |
| 案號 | 例 SP11505，無案號可省略括號 |
| 章節編號 | 第伍章、第陸章、第柒章皆可 |
| 工作項目數量 | 幾列依實際計畫調整 |
| 工作項目名稱 | 依採購需求／計畫內容撰寫 |
| 各項金額 | 依實際報價填寫 |
| 單價、數量、單位、說明 | 可保留或留空 |
| 計畫正文 | 背景、方法、時程等不限格式 |
| 主 PM、合約日期 | 文件可不寫；上傳後於系統**手動填寫** |

---

## 三、檔案與上傳

| 項目 | 建議 |
|------|------|
| 檔案格式 | `.docx`（最穩定），`.pdf` 次之 |
| 檔名 | 建議含計畫名稱，例 `115高雄農村加值_服務建議書.docx` |
| 上傳後 | 確認狀態為「已辨識 · N 個工作項目 · 合約 NT$XXX」 |
| 若失敗 | 先按 ✕ 清除，檢查經費表是否為 Word 表格，再重新上傳 |

---

## 四、常見錯誤

| 錯誤做法 | 結果 |
|----------|------|
| 經費表用 Excel 截圖貼上 | 常只抓到標案名稱，金額為 0 |
| 表頭改成「項目」「費用」但無對應關鍵字 | 抓不到工作項目 |
| 只有目錄寫「經費預算配置62」 | 抓不到（要有正文表格） |
| 計畫名稱沒有「」 | 標案名稱可能空白 |

---

## 五、完稿檢查表

- [ ] 封面計畫名稱有「」
- [ ] 有「委託專業服務案」或「委託勞務案」
- [ ] 正文有「經費預算配置」章節
- [ ] 經費表為 Word 表格（非圖片）
- [ ] 表頭有「服務細項」與「複價」
- [ ] 每列工作項目金額 ≥ 1,000，使用千分位
- [ ] 有「總金額」列
- [ ] 上傳測試成功
